A Field Guide for Sanat

How Claude Remembers You

Claude starts each day with a blank mind. This is how you hand it a note that says who you are and what you're working on — automatically, every time.

The One Idea

Claude has no memory between sessions. It has a desk.

Think of it as a brilliant assistant with total amnesia. Each time you start, the desk is empty — it knows nothing about you or your work until something is placed in front of it. Everything it can see right now is called its context.

A file called CLAUDE.md is the note you leave on that desk. Claude reads it the moment you begin, so it walks in already briefed. You never have to re-explain yourself.

The Craft

What to write in the note

Keep it short and true. The test for any line: would you have to tell Claude this again next time? If yes, write it down once.

✓ Worth writing

  • How you like to be talked to ("plain language, no jargon")
  • Who you are and what you do
  • What this project is and who it's for
  • Rules that always apply ("always show me a summary first")
  • A lesson learned the hard way

✗ Leave it out

  • Today's one-off request — just say it in the chat
  • Things that change every week — they go stale
  • Long documents — point to the file instead
  • Stuff that's already in a file Claude can open
  • Filler — every line takes up desk space

The Takeaway

Set the desk before you start.

One note about you at the top, one note about each project inside its folder. Claude reads them automatically and shows up already knowing what matters. Write it once; never repeat yourself again.
